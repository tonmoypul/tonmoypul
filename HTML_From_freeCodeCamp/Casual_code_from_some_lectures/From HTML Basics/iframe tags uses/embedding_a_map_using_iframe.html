Steps to follow while embeddig map

You can embed a map using an iframe in a few different ways depending on the map service:

Google Maps Go to maps.google.com and search for a location Click Share → Embed a map Copy the <iframe> code provided Paste it into your HTML
<iframe
src="https://www.google.com/maps/embed?pb=!1m18!..."
width="600"
height="450"
style="border:0;"
allowfullscreen=""
loading="lazy">
</iframe>

OpenStreetMap
Go to openstreetmap.org and navigate to a location
Click the Share icon (on the right sidebar)
Check Include marker if needed, then copy the HTML iframe code
<iframe
width="425"
height="350"
src="https://www.openstreetmap.org/export/embed.html?bbox=..."
style="border: 1px solid black" >
</iframe>

General iframe tips
width / height — set size in pixels or use 100% for responsive layouts
allowfullscreen — lets users expand the map
loading="lazy" — defers loading until the map is in view (good for performance)
style="border:0;" — removes the default border

Making it responsive
Wrap the iframe in a container div:
<div style="position: relative; padding-bottom:
56.25%; height: 0; overflow: hidden;">
<iframe
src="YOUR_MAP_LINK_HERE"
style="position: absolute; top: 0; left: 0;
width: 100%; height: 100%; border: 0;"
allowfullscreen
loading="lazy">
</iframe>
</div>
This gives you a 16:9 aspect ratio map that scales with the page width.
